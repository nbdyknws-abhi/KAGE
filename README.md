# KAGE | Cinematic Streetwear Landing Page

KAGE (影) is an ultra-premium, fictional Japanese-inspired streetwear brand. This repository contains the front-end landing page, showcasing a modern, cinematic techwear aesthetic. 

Built with React, TypeScript, Tailwind CSS v4, and Framer Motion.

## Features
- **Immersive Hero**: Dynamic cyberpunk aesthetic with a breathing neon visual and 3D animated geometric logo.
- **Cinematic Lookbook**: A 10-look editorial slider with high-quality imagery and hardware profiles.
- **Global Cart System**: End-to-end global cart state (powered by React Context API + LocalStorage) featuring a sleek sliding drawer and mock checkout flow.
- **Responsive Layouts**: Designed to be flawlessly responsive from mobile to desktop.
- **Animated Interactions**: Extensive use of Framer Motion for scroll reveals, hover states, and smooth modal pop-ups.

## Tech Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React (Icons)

## Getting Started

### Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

### Deployment
This project is fully deployment-ready for static hosts (Vercel, Netlify, Cloudflare Pages, GitHub Pages). 

- **Build Command:** `npm run build`
- **Output Directory:** `dist`
